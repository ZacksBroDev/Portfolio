Mile High Karate

Study & Training Platform for Martial Arts Students

Mile High Karate is a full-stack web application I developed to support martial arts students in learning and reviewing the forms required for their belt tests. The platform acts as an educational resource, offering belt-specific guidance, structured study tools, and a streamlined experience tailored for students of all levels.

Tech Stack:

  • Front End: React (with HTML, CSS, and JSX)
  • Back End: MySQL
  • Authentication: Auth0

Key Features:

  • Belt-specific study sections for different forms
  • Secure user login and session handling via Auth0
  • Responsive design with a clean, intuitive React interface
  • Dynamic data loading for personalized learning content

I designed and developed this project independently, focusing on creating a clean user experience backed by robust data handling and secure authentication. It demonstrates my ability to build a fully functional full-stack solution with modern web technologies.

Jonezen Performance

Automotive Performance Shop

Jonezen Performance is a fully responsive React-based website I developed to showcase and promote a high-performance automotive shop that specializes in building fast cars and offering custom vehicle services.

The site is designed to reflect the energy and quality of the brand while giving potential clients an easy way to explore services, view testimonials, and get in touch for bookings.

Key Features:

  • Built with React for a fast, responsive user experience
  • Integrated booking calendar so users can schedule appointments or consultations
  • Client testimonials section to build trust and credibility
  • Mobile-first, responsive design for smooth browsing across all devices
  • Contact form for quick inquiries and service requests

I designed and developed the entire site, focusing on clean visuals, clear calls to action, and a structure that supports future scalability (such as adding photo galleries or a blog).

The Futons

Band Website & Digital Presence

The Futons is a custom website I designed and built for a local independent band to help them establish an online presence and connect with their fans. The goal was to create a simple, stylish, and responsive site that reflects their sound and identity.

Tech Stack:

  • Front End: HTML, CSS, JavaScript (vanilla)

Key Features:

  • Music section with links to streaming platforms
  • Tour date section to display upcoming shows
  • Media gallery with photos and performance images
  • Contact form for booking and fan outreach
  • Responsive layout for seamless viewing on mobile and desktop

I handled all design and development aspects of the project, from layout and styling to interactivity and mobile optimization. The site gives the band a professional online presence while keeping things lightweight and easy to update.
