# Screenshots Folder

Add project screenshots and demo GIFs here to enhance your portfolio and README.

## Recommended Structure

```
screenshots/
  mhk/
    - dashboard.png
    - belt-system.png
    - video-player.png
    - demo.gif
  
  justmalikbeats/
    - homepage.png
    - checkout.png

    - demo.gif
  
  remoranotes/
    - notes-list.png
    - editor.png
    - sync-demo.gif
```

## Screenshot Best Practices

1. **Resolution:** Minimum 1920x1080 for desktop, actual device resolution for mobile
2. **Format:** PNG for static images, GIF (max 5MB) or MP4 for demos
3. **Content:** Show key features and user flows, not empty states
4. **Privacy:** Blur any personal data or test accounts

## Tools for Capturing

- **macOS:** Cmd+Shift+4 (screenshot), QuickTime (screen recording)
- **Windows:** Win+Shift+S (screenshot), Xbox Game Bar (recording)
- **GIF Creation:** [ScreenToGif](https://www.screentogif.com/) or [Gifski](https://gif.ski/)
- **Video to GIF:** [ezgif.com](https://ezgif.com/video-to-gif)

## Usage in README

![Demo](https://user-images.githubusercontent.com/yourusername/demo.gif)
